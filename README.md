# PenterepTools - ptvulnsearcher

Tool for searching Common Vulnerabilities and Exposures (CVE).

**Install**
```
pip install ptvulnsearcher
```

**Usage**
```

```

**Usage example**
```
ptvulnsearcher -s Apache v2.2
```

**Options**
```
-s     --search    <search>     Search keywords
-cve   --cve,      <cve>        Search specific CVE
-j     --json                   Output in JSON format
-h     --help                   Show help message and exit
-v     --version                Show version
```
